\doxysubsection{Detailed Description}
This class Is a transient data model class for your Tools within the \mbox{\hyperlink{classToolFramework_1_1ToolChain}{Tool\+Chain}}. If Tools need to comunicate they pass all data objects through the data model. There fore inter tool data objects should be deffined in this class.
